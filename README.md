🚀 PathPointer

PathPointer is a full-stack web app to track coding progress and stay consistent with problem solving.

🔹 Features

Google login

Profile setup (LeetCode ID, topics, language)

Problem recommendations

Progress tracking

Goal setting

🛠 Tech Stack

Frontend: React (Vite)

Backend: Spring Boot

Database: PostgreSQL

Auth: Google OAuth

🚧 Status

Working: login, profile, dashboard

In progress: recommendations fix, email system

🎯 Goal

Build a system that helps maintain daily coding consistency.
